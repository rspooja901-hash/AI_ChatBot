# Simple AI Chatbot

This is a beginner-friendly AI mini project developed using Python.
The chatbot interacts with users by responding to common messages such as greetings, questions, and exit commands.

## Features

* Greeting response
* Basic conversation replies
* User input handling
* Exit command
* Rule-based AI logic

## Tech Used

* Python

## How to Run

```bash
python chatbot.py
```
